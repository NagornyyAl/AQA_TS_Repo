# Lesson 11 Task

Independent Vitest project based on the `lesson9` bird example.

## What is tested

- `vi.fn()` function mocks.
- `Mocked<T>` object mocks.
- `vi.spyOn()` with replaced method implementation.
- Isolated service tests with mocked repository and notifier objects.
- Injected movement strategy mocks for bird classes.

## Commands

```bash
npm run typecheck
npm test
```
