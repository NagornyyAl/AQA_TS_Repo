# Lesson 17: Selenium WebDriver

Hotline.ua tests rewritten from lesson 16 with Page Objects, Mocha, Chai and Selenium WebDriver.

## Run

```bash
npm install
npm test
```

By default Chrome runs headless. Use `HEADLESS=false npm test` in a Unix-like shell or `$env:HEADLESS='false'; npm test` in PowerShell to see the browser.
